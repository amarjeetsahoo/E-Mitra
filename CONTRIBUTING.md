# Contributing to E-Mitra

Thank you for your interest in contributing to E-Mitra! This project aims to empower migrant workers and labourers with accessible, voice-first legal knowledge.

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project.
2. **Create a Branch**: Make a new branch for your feature or bug fix (`git checkout -b feature/your-feature-name`).
3. **Make Changes**: Implement your changes. Please ensure your code follows the existing style and is well-documented.
4. **Test**: Ensure the application still works seamlessly, especially the voice interactions and Elasticsearch queries.
5. **Submit a Pull Request**: Push your branch to your fork and submit a PR to the main repository.

## Antigravity (AI Pair Programmer)

This project was developed with the assistance of **Antigravity**, an advanced agentic coding assistant by Google DeepMind. If you are using Antigravity or a similar AI tool to contribute:
- Please ensure the AI-generated code is thoroughly reviewed and tested.
- Do not check in sensitive configuration files (`.env`, `skills-lock.json`, etc.).
- Clearly describe the architectural choices made by the AI in your PR description.

## Setting Up Locally

Please refer to the `README.md` for full instructions on setting up the Elasticsearch index and the Node.js / React application locally.

Thank you for helping make legal rights accessible to everyone!
